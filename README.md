# Memory Card Matching Game

A responsive Memory Card Matching Game built with React. Match all card pairs in the least number of moves and shortest time.

##  Live Demo

https://your-vercel-link.vercel.app

##  GitHub Repository

https://github.com/Vishalaury/memory-card-matching-game

##  Features

- 4×4 Memory Card Game (16 Cards / 8 Matching Pairs)
- Random Card Shuffle
- Flip Two Cards at a Time
- Match & Auto Flip Back Logic
- Move Counter
- Game Timer
- Congratulations Screen
- Restart Game
- Responsive Design
- Card Flip Animation
- Difficulty Levels (Easy, Medium, Hard)
- Best Score Tracking using Local Storage

## 🛠️ Tech Stack

- React
- JavaScript (ES6+)
- CSS3
- Framer Motion
- React Icons
- Vite

##  Installation

Clone the repository:

```bash
git clone https://github.com/Vishalaury/memory-card-matching-game.git
```

Go to the project folder:

```bash
cd memory-card-matching-game
```

Install dependencies:

```bash
npm install
```

Start development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```
